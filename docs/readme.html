Information

Getting started:
  • install.html - Installation notes and initial configuration
  • gameoperator_intro.html - how to configure the lobby with details of a new game; also describes many of the main data structures.
  • lobbyservice_software_guide.html - description of the lobby service code and respository structure and some of the main dependencies
  • lobbyservice_client_API.html - description of the client web API

Design-related documentation:

  • lobbyservice_overview.pptx - Background and whole-system overview
  • lobbyservice_design_notes.html - (old) brief design notes; not completely consistent with current implementation
  • lobbyservice_interaction_notes.html - (old) design notes on how client/lobby interactions might work; only partially consistent with current implementation
  • lobbyservice_next_steps.html - design notes on areas of extension / development.
